应用软件

基础软件​

  1. 搜狗输入法
  2. Chrome
  3. Logitech Options

开发工具​

  1. Dash：Mac专属文档查询工具。
  2. 数据库：我个人开发使用MySQL以及Sequel Pro， 安装配置以及使用指南请参考这篇文章。
  3. Charles：抓包/Mock工具，具体配置点击查看。

常用软件​

  1. Chrome
  2. Paste：剪切板增强工具。
  3. Divvy：窗口管理， 多/大显示器尤其实用。
  4. Snipaste：截图工具。
  5. Hidden Bar：隐藏状态栏图标。
  6. IINA：多媒体播放，mpv内核，开源软件，快速迭代中，强推。
  7. Typora：MarkDown写作。
  8. Spark：邮件客户端。
  9. 欧路词典。
  10. Logitech：罗技设备，离不开。
  11. PDF Expert。
  12. ClearView：阅读工具，略贵，但完美支持mobi，epub，pdf。
  13. The Unarchiver：解压缩
  14. NTFS For Mac：读写 NTFS 文件格式的存储设备。
  15. HandShaker：情怀工匠精神锤出品，mac上管理安卓手机文件一把好手.
    • 更新：锤子很久不更新了，可改用Airdroid 以及 Android File Transfer
  16. Welly: 论坛灌水， 北邮人 && 水木网友你们懂的。
    brew cask install welly
  17. Noizio：白噪音软件，可以组合多种常见白噪音，强推。
  18. Take a break：每20分钟站起来走走
  19. Rescue Time：日常行为管理，可以分析出日常使用电脑时间分配.细化到软件/网站层面，对于提升工作效率还是很有意义的。

美化​

这类软件属于没有无所谓，有了以后心情会好一些，所以推荐可以尝试下。

  1. Irvue：自动获取 Unsplash 上的高质量无版权图片作为壁纸，并提供了其它一系列相关的功能，如对图片点赞、下载、加入黑名单等。
    • 更新：升级10.14后，改用macOS自带的动态壁纸了。
  2. Wallpaper Wizard 2：也是壁纸应用，特点是提供一个很大的壁纸库用以挑选，之后更换自己事先挑选好的壁纸。
  3. Fliqlo：屏保，极简时钟。
  4. rasmusnielsen：屏保，拟物表盘。

其他日常使用技巧​

  1. 微信免登陆

    git clone https://github.com/Sunnyyoung/WeChatTweak-macOS.git
    cd WeChatTweak-macOS
    sudo make install

    重启微信后，修改Preference。

  2. 清除软件卸载残留

    # Mac完全卸载软件
     ~/Library/Application Support/(应用程序名称)
     ~/Library/Preferences/(应用程序名称)
     ~/Library/Caches/(应用程序名称)

  3. 简单翻墙:下载可用hosts 替换/etc/hosts
