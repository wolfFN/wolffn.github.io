Data Type

常用类型​

整数数据，使用 INT 类型；
包含小数点的数据，使用 DOUBLE 类型。

长度不超过255个字符，使用 VARCHAR；
超过255个字符，使用 TEXT。

日期：DATE； 时间：TIME； 日期和时间：DATETIME。

在 MySQL 中，日期与时间类型的数据需要使用单引号 ' 或者双引号 " 括起来。

  • 日期必须以 YYYY-MM-DD 的格式输入。
  • 时间必须以HH：MM：SS的格式输入。
